# 🌍 LST Delhi Analysis

## 📌 Project Overview
This project focuses on analyzing Land Surface Temperature (LST) of Delhi using Landsat 8 satellite data. The aim is to understand temperature variations across different land cover types.

## 🎯 Objectives
- To calculate Land Surface Temperature (LST)
- To analyze temperature variation in urban areas
- To study the impact of vegetation and built-up areas on temperature

## 🛰️ Data Used
- Landsat 8 (Level 1 Collection 2)
- Study Area: Delhi
- Time Period: May–June (High temperature season)

## 🧠 Methodology
The following steps were performed:
1. TOA Radiance calculation
2. Brightness Temperature (BT)
3. NDVI calculation
4. Proportion of Vegetation (PV)
5. Final LST calculation

## 🛠️ Tools & Software
- QGIS
- Remote Sensing Techniques

## 📊 Results
- Higher temperature observed in built-up areas
- Lower temperature in vegetation and water bodies
